Ritchey Permissive License v16 is the 16th release of Ritchey Permissive License. It is a permissive license, designed for open works, and open source works. The project has changed dramatically over the years, so beyond the basic purpose of the license, it can be important discuss versions as completely separate works. For example, there are dramatic differences in language, and features between the original release, and the current one.

Version 16 has several features such as: defined terms, sections, imposed restrictions, jurisdiction binding, and a disclaimer.

While versions of the license have been in use for years by the author, it remains a an obscure license within the open source community.
