Ritchey Permissive License is an open license developed by James Ritchey. The first release was made in 2015. As of 2023, there have been 16 releases.

The most notable releases are:

-Ritchey Permissive License v11, which was rejected by the OSI as not conforming to the OSD.
-Ritchey Permissive License v12, which as rejected by the OKF community.
-Ritchey Permissive License v4, which was the first release to the use the name Ritchey Permissive License.
-Ritchey Permissive License v16, which was the first release to adopt sectioning.

The license is primarily used, and intended for open-source projects, but the terms themselves do explicitly mention source code.
